# Streamlit GUI V2 系統整合說明

## 更新概要

Streamlit GUI 已完全整合 V2 系統功能,支持在回面中輕鬆切換 V1/V2 模式。

### 2. 🜢 V2 模型訓練標籤

**位置**: 第 4 個標籤 (僅當 V2 啟用時顯示)

**功能**:
1. **訓練模式選擇**
   - 🚀 快速測試 (30-60分鐘)
   - 🏆 完整訓練 (2-4小時)
   - 💡 自定義

2. **特徵配置**
   - 啟用/關閉進階特徵 (25個)
   - 啟用/關閉 ML 特徵 (10個)
   - 實時顯示特徵數

3. **實時訓練狀態**

   - 實時日誌輸出

**使用流程**:
```

   ↓

   ↓
3. 選擇訓練模式 (快速/完整/自定義)
   ↓
4. 配置特徵選項
   ↓
5. 點擊 "🚀 開始訓練"

6. 等待訓練完成
   ↓
7. 查看結果分析
   ↓
8. 前往回測頁面
```

**增強功能**:

**使用示例**:

  🜢 本次回測使用 V2 特徵 (44-54個)

## 使用指南

### 快速開始

#### 步驟 1: 啟動 Streamlit

```bash
streamlit run main.py
```

#### 步驟 2: 選擇 V2 模式

1. 在左側邊欄找到 "🆕 版本選擇"
2. 選擇 "🜢 V2 - 進階版 (44-54特徵)"
3. 看到 "✨ V2 系統已啟用"

#### 步驟 3: 訓練 V2 模型

1. 點擊 "🜢 V2 模型訓練" 標籤

1. 點擊 "📊 策略回測" 標籤

3. Long/Short 模型都選擇 "🜢 V2 (推薦)"
4. 選擇剛才訓練的模型
5. 配置參數 (使用預設值)
6. 點擊 "🚀 執行標準回測"

1. 記錄 V2 的 Profit Factor
2. 切換到 V1 模式 (侧邊欄)
3. 點擊 "📊 策略回測"
4. 選擇 V1 模型執行回測
5. 對比兩個版本的結果

## 注意事項

### 1. 版本相容性

✅ **V1 和 V2 可以共存**
- V1 模型: `catboost_long_20260224_120000.pkl`
- V2 模型: `catboost_long_v2_20260224_133045.pkl`
- 不會互相影響

⚠️ **不要混合使用**
- Long 用 V2,Short 用 V1 → 會顯示警告
- 建議使用相同版本

### 2. 特徵工程

🔄 **自動選擇**
- GUI 會根據模型版本自動選擇對應的 feature engineer
- V1 模型 → `FeatureEngineer`
- V2 模型 → `FeatureEngineerV2`
- 無需手動介入

### 3. 性能影響

⏳ **訓練時間**
- V1 快速訓練: 10-20 分鐘
- V2 快速訓練: 30-60 分鐘
- V2 完整訓練: 2-4 小時

💧 **內存使用**
- V1: ~2GB
- V2 全特徵: ~6GB
- V2 節省模式: ~4GB (關閉進階特徵)

### 4. 建議工作流

💡 **測試階段**
```
1. V1 快速訓練 (10分鐘)
   ↓
2. V1 回測基線 (PF = 1.22)
   ↓
3. V2 快速訓練 (30分鐘)
   ↓
4. V2 回測驗證
   ↓
5. 如果 V2 > V1 → V2 完整訓練 (3小時)
```

🚀 **生產階段**
```
1. V2 完整訓練
   ↓
2. V2 回測驗證 (PF > 1.4)
   ↓
3. Paper Trading 7天 (PF > 1.2)
   ↓
4. Live Trading
```

## 常見問題

### Q1: 不小心點到 V1 標籤,會影響 V2 嗎?

**A**: 不會。V1 和 V2 是獨立的,可以同時使用。

### Q2: 如何知道回測使用的是 V1 還是 V2?

**A**: 回測結果頂部會顯示:
- "🜢 本次回測使用 V2 特徵 (44-54個)"
- "🔵 本次回測使用 V1 特徵 (9個)"

### Q3: V2 訓練太慢怎麼辦?

**A**: 使用快速測試模式:
1. 選擇 "🚀 快速測試"
2. 或關閉進階特徵 (從 54 降到 34 個)
3. 或關閉 ML 特徵 (從 54 降到 44 個)

### Q4: 如何切回 V1 模式?

**A**: 
1. 在侧邊欄選擇 "🔵 V1 - 基礎版"
2. 標籤會自動恢復到 5 個 (移除 V2 標籤)

### Q5: V2 特徵會影響舊 V1 模型嗎?

**A**: 不會。V1 模型只能用 V1 特徵,GUI 會自動辨識。
