# Crypto Reversal Trading System V1\n\nAn advanced cryptocurrency trading system specialized in detecting and trading reversal signals using order flow imbalance and liquidity zone analysis.\n\n## Strategy Overview\n\nThis system implements a unique reversal trading strategy that combines:\n\n1. **Order Flow Imbalance (OFI) Detection** - Identifies extreme buying/selling pressure imbalances that often precede reversals\n2. **Liquidity Zone Analysis** - Detects liquidity sweeps and smart money operations at key price levels\n3. **Market Microstructure** - Analyzes candlestick patterns and internal price behavior\n4. **ML Signal Validation** - Uses machine learning to filter false signals and validate reversal probability\n\nUnlike traditional indicator-based strategies, this approach focuses on market microstructure and order flow dynamics for more accurate reversal prediction.\n\n## Features\n\n- Modular architecture with clean separation of concerns\n- Real-time data fetching from Binance API for backtesting\n- Historical data support via HuggingFace datasets\n- Comprehensive OOS (Out-of-Sample) validation\n- Interactive Streamlit GUI with 5 main tabs\n- Support for leverage trading (1x-20x)\n- Detailed backtest with realistic fees and slippage\n- Risk management with ATR-based stop loss and take profit\n\n## Project Structure\n\n```\nreversal_strategy_v1/\n├── core/                     #
